# Web PDF to JPG Converter

一个基于网页的PDF转JPG工具，允许用户将PDF文件转换为JPG图片，并可选择将所有图片合并为一张大图。

## 功能特性

- 将PDF文件的指定页面转换为JPG图片
- 支持自定义页面范围（起始页和结束页）
- 可设置图片质量（DPI）
- 可设置输出图片的文件名前缀
- 可选择将所有图片合并为一张大图（竖向或横向）
- 完全在浏览器中运行，无需上传文件到服务器
- 支持批量下载生成的图片
- 纯前端实现，数据安全性高

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- [PDF.js](https://mozilla.github.io/pdf.js/) - 用于PDF渲染的JavaScript库

## 使用方法

### 本地运行

1. 克隆或下载此仓库
2. 启动HTTP服务器（如Python的内置服务器）

```bash
cd web-pdf-to-jpg
python3 -m http.server 8000
```

3. 访问 `http://localhost:8000`

### 部署到服务器

将整个项目目录上传到任何支持HTTP服务的Web服务器即可。

## 使用指南

1. 选择要转换的PDF文件
2. 设置页面范围（可选，留空表示所有页面）
3. 设置图片质量（DPI）
4. 设置输出图片的前缀名
5. （可选）勾选“创建合并图片”以将所有图片合并为一张大图
6. 选择合并方向（竖向或横向）
7. 点击“开始转换”
8. 查看结果并下载图片

## 注意事项

- 所有PDF处理都在客户端浏览器中进行，文件不会上传到任何服务器
- 建议使用现代浏览器以获得最佳体验
- 处理大文件或页数较多的PDF时可能需要较长时间，请耐心等待
- 图片质量越高，生成的文件越大，转换时间也可能越长
